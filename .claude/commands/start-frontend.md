# Start Frontend Dev Server

Start the Vite development server for the React frontend.

```bash
cd /Users/consultadd/Downloads/Thunderbird-SolarPumps/frontend && npm run dev
```

Run this in the background. After it starts, tell the user:
- Local URL: http://localhost:5173
- To share publicly with the client, also run `/ngrok-url` (ngrok must already be running)
- The backend must be running on port 8000 for calculations to work
